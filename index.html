Simulation for a specific manufacturing process (Electrical Discharge Machining)

Design and programming of the whole simulation. The results help understand the mechanisms in tool wear and determine the machining parameters for better performance. See publications for what is achieved.

Inkjet printer designed for liquid metal

Design and programming of a functional inkjet printer for liquid metal with user interface. The printer can efficiently print any input dot pattern on glass, PDMS or other flat surface with input parameters.

Direct writing printer for liquid metal

Design of programming of a funtional direct writing printer for liquid metal, especially eGaIn with user interface. The printer can direct write eGaIn following an input trace. More tests are being made on the printer.
